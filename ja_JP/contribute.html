GORMにコントリビュートする

より良いGORMを提供する手助けをすることが可能です！あながができることはたくさんあります：

GORMをほかの人達に知らせる：

  • GORMをソーシャルなチャネルから世界にシェアする、もしくは単純に興味がありそうな人に教える
  • Stack Overflow 、 Github Issues Github Issues そしてチャットルーム 上での新参者の疑問を解決するのを助ける
  • チュートリアルを書く、ビデオを撮る、等…(リストに追加するのをお忘れなく)
  • GitHub上でGORMのリポジトリにStarを付けたり WatchすることでGORMを宣伝するのを助ける

ドキュメントを書く、執筆する

  • 新しいコンテンツを書いたり、既存の情報を修正することで、GORMのドキュメントをより良くする

このサイトを翻訳する

https://translate.gorm.ioを使ってこのサイトを翻訳することが可能です

GORMをあなたの言語に翻訳するには、 https://translate.gorm.ioに追加するためのGithub issuesへの投稿が必要です。

プログラム

  • 問題やGithub Issuesで同意を得た機能を修正するために、Githubにプルリクエストを作成します
  • GORM用のオープンソースのプラグインを作成します。 (それらをthe listに追加するのをお忘れなく)
