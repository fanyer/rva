迁移

自动迁移

自动迁移你的模型，使之保持最新状态。

警告： 自动迁移 只会 创建表、缺失的列、缺失的索引， 不会 更改现有列的类型或删除未使用的列，以此来保护您的数据。

其它迁移工具

GORM 的自动迁移在大多数情况下都会正常工作，但如果你需要更严格的迁移工具， GORM 提供了通用 DB interface ，这可能对你有帮助。

参考 通用 Interface 获取详情。
