贡献 GORM

为了 GORM 更好的发展，您可以在在下面的事情上提供您的帮助：

帮助他人了解 GORM

  • 通过社交渠道与世界分享 GORM, 或者告诉那些可能感兴趣的人。
  • 帮助新来者解决 Stack Overflow 、 Github Issues 和 聊天室 的问题
  • 编写教程、录制视频等。(不要忘记将它们添加到 这个列表 中)
  • 通过 加星 和 关注 GORM 来帮助在 GitHub 推广 GORM

编写文档

  • 通过编写新内容、更正现有内容来让 GORM 的文档更好

翻译此网站

你可以通过 https://translate.gorm.io 来翻译本网站

如果你想将本站翻译到一个新语言，请在 Github 提交一个 issue，然后我们会将其添加到 https://translate.gorm.io 中

编程

  • 修复 Github Issues 中的问题或完成讨论后新功能，并提交 Pull Request 到 Github
  • 编写开源插件。(不要忘记将它们添加到 这个列表 中)
